# Images Directory

Add your image files to this folder. The website supports:
- JPG/JPEG files
- PNG files
- WebP files
- GIF files

Simply place your image files here and update the `index.html` file with the proper image filenames and descriptions.

Example:
```html
<div class="image-card">
    <img src="images/your-image.jpg" alt="Your Image Description">
    <div class="image-info">
        <h3>Your Image Title</h3>
        <button class="favorite-btn" onclick="toggleFavorite(this)">â™¡</button>
    </div>
</div>
```
